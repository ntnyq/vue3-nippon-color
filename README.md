# vue3-nippon-color

![CI](https://github.com/ntnyq/vue3-nippon-color/workflows/CI/badge.svg)

A clone of [nippon-color](https://github.com/ssshooter/nippon-color).

## Commands

```bash
# install dependencies
pnpm install

# start dev server
pnpm run dev

# build for production
pnpm run build

# lint code style
pnpm run lint
```
